Machine Learning, What Is It?
No doubt, you've been hearing a lot about machine learning in recent years. What is it? And do we expect sentient robots to rise up and take over the world like Skynet?
Probably not. Simply put, machine learning are techniques, theories, and methods that grew out of early work on artificial intelligence and has seen a resurgence in recent years due to new capabilities in faster and more powerful computing resources.
That said, what do people even use use machine learning for?
Database mining from large datasets that have grown from the web and various automation technologies: i.e. web click data, medical records, biology, engineering, etc.
Applications that can't be programed by hand, increasingly we are seeing the need for applications that can handle ambiguity: i.e. chatbots, virtual agents, handwriting recognition, most natural language processing (NLP), computer vision, etc.
Self-customizing programs, also known as recommendation engines, utilize techniques in unsupervised learning and are found in services like: Amazon, Netflix product recommendations, etc.
Objectives
While there are a lot of resources out there to help you learn more about machine learning, this web page is focused on giving you practical code examples and explanations on how to work with one of the most well-known machine learning services, the Watson Natural Language API.
Before being acquired by IBM, the Watson Natural Language API was known as AlchemyAPI and one of the leading machine learning services in the market from 2011 to 2016, rivaling any Google or Facebook solution. While the competition has caught up, learning how to use this API will give any aspiring machine learning enthusiast a solid foundation in the potential benefits and applications of machine learning technologies.
